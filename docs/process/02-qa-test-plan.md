## Key Interactions to Verify

- Reload the page → verified/unverified status persists (no re-verify drift)

## Edge Cases
- Quote containing ':' and single ']' parses correctly (split on first '::')
- Quote with HTML entities (&, <) still verifies (html.unescape)
- Same clause cited twice → one footer entry, number reused
- Unknown/fabricated doc_id → shown flagged, non-clickable "unknown source"
- Model emits zero markers → honest amber (verify demo question cites reliably)

## Critical Paths
- Ungrounded answer renders amber "Unverified", never a confident uncited answer
- Upload summary is NEVER flagged amber (D2)
- sources_cited reflects verified count, not regex hits (regression vs old test)
